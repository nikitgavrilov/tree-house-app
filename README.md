## Использование

Установите node_modules в директориях client и server с помощью команды:
```sh
cd client
npm i
cd ../
cd server
npm i
```
Выполните команду npm start после установки node_modules. Это запустит Development сервер для backend и frontend 
```sh
npm start
cd ../
cd client
npm start
```
